## Auto Proxy Scraper

Scrapes free proxy lists from [https://free-proxy-list.net/](https://free-proxy-list.net/) and saves them in a markdown file.

## Usage (Local)

If you want to run this scraper on your local machine:

1. **Install dependencies:**

   ```bash
   npm install
   ```

2. **Run the scraper:**

   ```bash
   npm start
   ```

3. **Check the results:**
   The scraped proxies will be saved in `proxies.md` in the root directory of this project.
